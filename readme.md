TP°4 — Mini Aplicación Web

Este proyecto es una mini aplicación web desarrollada con JavaScript moderno (ES6+) que integra manipulación del DOM, eventos y consumo de APIs externas.

El objetivo es aplicar conceptos como:
Arrow functions
Arrays (map, filter, reduce, find)
Manipulación del DOM
Eventos
Fetch API con async/await

Paginas del TP

Productos
Sistema de filtrado dinámico de productos:
Filtrado por categoría
Filtrado por precio máximo
Mostrar solo productos en stock
Búsqueda por nombre en tiempo real

Todo
Aplicación de lista de tareas:
Agregar tareas
Marcar como completadas
Eliminar tareas
Contador dinámico de tareas pendientes
Validación para evitar tareas vacías

---

Pokemon Api
Consumo de API externa (PokéAPI):
Fetch con async/await
Estado de carga ("Cargando...")
Manejo de errores
Renderizado dinámico en tarjetas

Buscador
Búsqueda en tiempo real
Mínimo 3 caracteres
Mensajes de estado (buscando, sin resultados, error)

Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox y Grid)
- JavaScript ES6+
- Fetch API
- API pública: PokéAPI
